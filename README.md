# Korea Income & Welfare

Proyecto de analisis y modelado sobre ingreso y bienestar en Corea. Ahora el repositorio queda mejor posicionado como caso de portfolio: con objetivo explicito, dependencias claras y un marco de lectura mas profesional.

## Objetivo analitico

Estudiar como variables socioeconomicas, especialmente educacion, se relacionan con el ingreso y el bienestar.

## Valor del caso

- Muestra manejo de datos sociales de gran escala.
- Combina EDA, modelado y lectura aplicada.
- Puede presentarse tanto en clave academica como de consultoria.

- `Korea_Income_and_Welfare.csv`
