# Korea Income & Welfare

Proyecto de analisis y modelado sobre ingreso y bienestar en Corea.

## Objetivo

Estudiar la relacion entre variables socioeconomicas, educacion e ingreso a partir de un dataset de bienestar e ingresos.

## Herramientas

- Python
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- seaborn
- scipy
- scikit-learn
- bokeh

## Conclusion

El repo combina exploracion y modelado en una base util para seguir refinando insights y mejorar la presentacion ejecutiva del caso.
