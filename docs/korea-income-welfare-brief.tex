\begin{center}
\large Education premiums, predictive benchmarking, and distributional heterogeneity in Korean welfare microdata
\end{center}

\begin{abstract}
This brief studies whether education still carries measurable income value once household size, age, region, gender, religion, and survey year are added to the analysis. The repository combines descriptive summaries, non-parametric tests, machine-learning benchmarking, semilog OLS with robust inference, and quantile regression. The central result is that education has a clearly positive average premium, but that premium is not constant across the income distribution. Machine learning improves pure prediction materially, while the econometric layer explains what those predictive gains mean.
\end{abstract}

\section{Question}

The project asks three linked questions:

\begin{enumerate}[leftmargin=1.5em]
  \item Does education show a strong and statistically meaningful relationship with income in Korean welfare microdata?
  \item How much does predictive performance improve once richer demographic and household information is added?
  \item Is the education premium constant, or does it vary across lower-income and higher-income parts of the distribution?
\end{enumerate}

The analysis is intentionally descriptive and predictive rather than causal. The goal is to measure signal, compare model behavior honestly, and explain the income premium in a way that is useful for analytics, labor-market research, and public-policy storytelling.

\section{Data and empirical design}

The raw dataset contains 92,857 survey records. After removing invalid income observations, restricting the age range to 18 through 90, and keeping rows with usable education information, the main analysis sample contains 89,935 observations. A standard interquartile-range trim on income yields a final modeling sample of 86,313 observations.

The workflow has five layers:

\begin{enumerate}[leftmargin=1.5em]
  \item Descriptive EDA across education, region, and survey year.
  \item Formal distribution tests using Kruskal-Wallis, Spearman rank correlation, and Mann-Whitney U.
  \item Predictive benchmarking with education-only linear regression, multivariable linear regression, Elastic Net, Gradient Boosting, and a tuned Random Forest.
  \item A semilog OLS earnings equation with centered age terms and HC3 robust standard errors.
  \item Quantile regression at the 25th, 50th, and 75th percentiles to test heterogeneous education returns.
\end{enumerate}

\section{Diagnostics and interpretation}

The semilog specification is not used blindly. Heteroskedasticity is present, so the main OLS table is reported with robust errors. Centering the age polynomial keeps VIF values in a low range and stabilizes the nonlinear life-cycle profile. RESET rejects the idea that a single simple linear specification is fully sufficient, which is exactly why the repository pairs the OLS layer with machine learning and quantile regression instead of presenting a single coefficient table as the whole story.

Taken together, the repository supports a clear conclusion: education has a strong positive average association with income, but that return is heterogeneous across the distribution. Richer socioeconomic context improves prediction substantially, yet interpretation still depends on the econometric layer.
